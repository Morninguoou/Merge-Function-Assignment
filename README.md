# Merge TypeScript 

Implementation of a merge function that combines three sorted arrays into a single ascending array.

## Rules
- collection_1 and collection_3: sorted ascending
- collection_2: sorted descending
- result must be sorted ascending
- no sort function allowed

## How to run

1. Clone the repository

    ```bash
    git clone https://github.com/Morninguoou/Merge-Function-Assignment.git
    ```

    ```bash
    cd merge-typescript
    ```

2. Install the dependencies

   ```bash
   npm install
   ```

3. Build the project

    ```bash
    npm run build
    ```

4. Run the tests

   ```bash
   npm run test
   ```
